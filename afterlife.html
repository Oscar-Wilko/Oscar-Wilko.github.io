About The Game

This assignment required me and 4 other students to work together to recreate an old game called 'Afterlife' first within Unity and then solely in DirectX11. The unity project enabled us to layout the foundations of what the game entaled and how we should or shouldn't produce it when making it in DirectX. The main difficult was the DirectX11 creation which forced our group to make every system, including back-end functionality like cameras, UI, and inputs. I was involved with a lot of the back-end systems surrounding the in game population, advisors, stats graph, and more. An important note is the windows software that I had worked on with another member of the group, which is programmed in XAML. It created an interface that allowed for custom dialogue within the game so that 'future game designers' could utilise it and produce content and a quickened pace.
