About The Game

Golf Mountain is my first videogame that I had ever made. It is a Foddian inspired game in which the player hits a golf ball up a 2D platforming mountain to reach the end. This project allowed me to better understand the workflow and capabilities of the Unity software. Which taught me Object-Oriented-Programming, game loops, player inputs, and much more...
